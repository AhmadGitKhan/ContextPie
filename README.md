# ContextPie
Context Sensitive Pie Menu for Blender, following an ancient Mayan pie recipe.

SHIFT + RIGHT MOUSE:
    Different functions for Object, Sculpt & Edit Mode based on selection

RIGHT MOUSE:
    Mode selection
